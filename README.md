# Sightseeing

Sightseeing is a sample application for collecting and listing data about interesting places around the world. 

## Technologies

* ASP.NET Core 3.1
* Entity Framework Core
* MS SQL
* CQRS 
* Feature folders and vertical slices
* MediatR
* AutoMapper
* Fluent Validation
* Scrutor
* Serilog
* JWT authentication
* Grpc
* Moq
* xunit
* Fluent Assertions
* Health Checks
